Iteritas Project Manifest

Iteritas is a modern travel exploration and booking prototype built with high-performance web technologies and Generative AI.

🚀 Tech Stack

Next.js 15 (App Router) React 19 TypeScript Tailwind CSS Shadcn UI Genkit AI Google Gemini 1.5 Flash Lucide Icons

🗺️ Route Map

  • / - Search: AI-powered natural language travel discovery.
  • /for-you - For You Feed: Curated recommendations and YouTube Reels "Explore" section.
  • /traverse - Traverse: Immersive travel video gallery using YouTube embeds.
  • /trips - My Trips: User's saved adventures and bookings.
  • /reviews - Reviews: Community feedback and travel experiences.
  • /profile - Profile: User settings, dark/light mode toggle, and "About Us" information.
  • /login - Authentication: Onboarding flow for new users.

🤖 AI Capabilities (Genkit)

  • Dynamic Search: Interprets vague user queries (e.g., "warm beach in December") using LLM reasoning to suggest specific flights, hotels, and experiences.
  • Personalized Recommendations: A backend flow designed to ingest user preferences and past behavior to generate custom travel itineraries.
  • Video Generation (Ready): Implementation exists for Veo models (requires GCP billing) for creating AI travel cinematics.

🎨 Design System

The app uses a "Space Grotesk" headline font paired with "Inter" for body text. The theme is controlled via HSL variables in src/app/globals.css, supporting seamless dark and light mode transitions via next-themes.
