# **App Name**: Iteritas

## Core Features:

- Home Page: The application's home page.
- "For You" Feed: Algorithm-driven tab with a compass logo interleaved video content, flight deals, restaurant recommendations, and hotel previews.
- Instant Booking: Allow users to book flights, hotels, and experiences with a seamless 3-tap booking process.
- Dynamic Search: Implement a search overlay that supports natural language queries. The LLM will use reasoning to decide which results best match the search using data like time of year, common search phrases and available flights.
- Personalized Recommendations: Use machine learning to provide hyper-personalized recommendations based on user data.
- Trip Boards: Enable users to save items to specific trip boards.
- Social Sharing: Allow users to directly share items and trips to Instagram/Snapchat.
- User Authentication: Implement user authentication via Firebase Auth (Google, Apple, and Phone number login).

## Style Guidelines:

- Background color: Dark gray (#121212) for a default dark mode experience.
- Primary color: Electric Lime (#CCFF00) to convey energy and excitement.
- Accent color: Hyper-Violet (#6C2DC7), analogous to electric lime but distinct, for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, with 'Inter' (sans-serif) for longer text
- Custom icons with a geometric style, in electric lime and hyper-violet.
- Use a "Bento Box" grid layout mixed with full-screen vertical swipe cards for the "For You" feed.
- Implement a floating bottom navigation bar.
- Subtle animations and transitions to create a seamless and engaging user experience.
